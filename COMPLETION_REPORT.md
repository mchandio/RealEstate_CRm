## 🚀 BUILD SYSTEM CAPABILITIES

### What The System Does

**Automated 6-Step Build Process**

```
Step 1: Check Requirements (5 sec)
└─ Verify Python, PyInstaller, PySide6, Inno Setup

Step 2: Clean Old Builds (2 sec)
└─ Remove build/, dist/, staging directories

Step 3: Build Desktop Application (10-15 min)
└─ Creates: RealEstateCRM_Qt.exe (150-200 MB)
   - Full PySide6 Qt framework
   - All CRM features
   - SQLite database support
   - Single executable or folder

Step 4: Build LAN Web Server (5-8 min)
└─ Creates: RealEstateCRM_LAN_Server.exe (100-150 MB)
   - FastAPI web server
   - Multi-user support
   - Web interface on port 6090
   - Database integration

Step 5: Stage Files (1 min)
└─ Copies executables and resources to staging area

Step 6: Create Windows Installer (2-3 min)
└─ Creates: RealEstateCRM_v2.1.0_Setup.exe (200-400 MB)
   - Inno Setup compilation
   - LZMA2 Ultra compression
   - Modern UI wizard
   - Ready for distribution

TOTAL BUILD TIME: 20-35 minutes (first build)
SUBSEQUENT BUILDS: 15-25 minutes (uses cache)
```

### Individual Build Steps

You can run each step separately:
- `python build_installer.py check` - 5 seconds
- `python build_installer.py build` - 15-20 minutes
- `python build_installer.py stage` - 1 minute
- `python build_installer.py installer` - 2-3 minutes

### Change Version
```python
# Edit: build_installer.py
class Config:
    VERSION = "2.2.0"  # Change this
```

### Change Company Name
```python
# Edit: build_installer.py
class Config:
    APP_NAME = "Your App Name"
    PUBLISHER = "Your Company"
```

### Add Files to Installer
```ini
# Edit: RealEstateCRM_Setup_Professional.iss
[Files]
Source: "path\to\file"; DestDir: "{app}\folder"; Flags: ignoreversion
```

### Change Installation Directory
```ini
# Edit: RealEstateCRM_Setup_Professional.iss
DefaultDirName={localappdata}\Programs\Your Company\Your App
```
