## 🚀 Getting Started

### Installation
1. Run the application:
   ```bash
   python -m CRM
   ```

2. The login dialog will appear

### First Login
- **Username:** `admin`
- **Password:** `admin`

⚠️ **Important:** Change the default password after first login!

## 🔐 Login

### Logging In
1. Enter your username and password
2. Click **Login** or press Enter

### Account Lockout
- After **5 failed attempts**, your account will be locked
- Lockout duration: **30 minutes**
- Contact an admin to unlock your account

### Password Requirements
- Minimum 8 characters
- At least 1 uppercase letter
- At least 1 lowercase letter
- At least 1 number
- At least 1 special character (!@#$%^&*...)

## 🏠 Property Management

### Rent Availability
1. Click **Rent** tab
2. View available rental properties
3. Click **Add** to create new listing
4. Click **Edit** to modify existing listing
5. Click **Delete** to remove listing

### Sale Availability
1. Click **Sale** tab
2. View properties for sale
3. Same actions as Rent Availability

### Search & Filter
- **Keyword Search:** Type in search box, press Enter
- **Sort:** Select column from Sort dropdown
- **Date Range:** Set Start/End dates
- **Clear Filters:** Click Clear button

## 💼 Deal Tracking

### Creating a Deal
1. Select a property listing
2. Click **Add Deal** or similar action
3. Fill in deal details
4. Click **Save**

### Deal Stages
Properties move through stages:
1. Lead
2. Inquiry
3. Qualification
4. Follow-up
5. Meeting
6. Property Visit
7. Negotiation
8. Token
9. Booking
10. Installments
11. Documentation
12. Transfer
13. Registry
14. Closed

---

## 💰 Installment Tracking

### View Installments
1. Click **Installments** tab
2. View all installment plans
3. See payment status for each

### Add Installment Plan
1. Click **Add**
2. Select deal
3. Enter total amount
4. Enter number of installments
5. Enter start date
6. Click **Save**

### Record Payment
1. Select installment plan
2. Click **Record Payment**
3. Enter payment details
4. Click **Save**

---

## 🤝 Commission Management

### View Commissions
1. Click **Commissions** tab
2. View all commission records
3. See status (Pending/Approved/Paid)

### Add Commission Split
1. Select commission record
2. Click **Add Split**
3. Enter agent name
4. Enter split percentage
5. Click **Save**

### Approve Commission
1. Select commission record
2. Click **Approve**
3. Confirm approval

### Mark as Paid
1. Select approved commission
2. Click **Mark Paid**
3. Confirm payment

---

## 👥 User Management

### View Users
1. Click **Users** tab
2. View all user accounts
3. See status (Active/Inactive/Locked)

### Add User
1. Click **Add User**
2. Enter username
3. Enter password (must meet strength requirements)
4. Enter full name
5. Select role
6. Click **Save**

### Edit User
1. Select user
2. Click **Edit/Password**
3. Modify details
4. Click **Save**

### Toggle Active Status
1. Select user(s)
2. Click **Toggle Active**
3. Confirm action

### Unlock Account
1. Select locked user
2. Click **Unlock Account**
3. Confirm unlock

## 🔧 Tips & Tricks

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Enter` | Submit form/Search |
| `Escape` | Cancel/Close dialog |
| `Ctrl+A` | Select all rows |
| `Delete` | Delete selected |

### Pagination Controls
- **First:** Go to first page
- **Prev:** Go to previous page
- **Next:** Go to next page
- **Last:** Go to last page
- **Per Page:** Change rows per page (50/100/250/500/1000)

### Selecting Records
- **Single click:** Select one record
- **Ctrl+Click:** Select multiple records
- **Select All:** Click Select All button
- **Clear:** Click Clear Selection button

## ❓ Troubleshooting

### Login Issues
- **"Account locked"**: Wait 30 minutes or contact admin
- **"Invalid credentials"**: Check username/password
- **"User not found"**: Contact admin to create account

### Display Issues
- Resize window to fit content
- Check display scaling settings

### Performance Issues
- Reduce page size (fewer rows per page)
- Use specific filters instead of broad searches
- Close other applications
